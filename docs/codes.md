## 3. 権限・フラグ定義 (システム定数)

### 3.1 ユーザー権限レベル (`tz910_permission`)
| 値 | 権限名 | 備考 |
| :--- | :--- | :--- |
| -1 | Hidden | 画面上に表示しない |
| 0 | Staff | 一般スタッフ（閲覧制限あり） |
| 1 | Manager | 支配人・管理者（全データ閲覧可） |
| 2 | Admin | システム管理者（設定変更可） |

### 3.2 時間休業フラグ (`ta220.temp_closed`)
ビット演算による論理和で管理します。
* `1`: 朝休業
* `2`: 昼休業
* `4`: 夜休業
